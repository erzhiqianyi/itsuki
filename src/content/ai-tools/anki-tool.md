### ツール概要 (Tool Overview)
このツールは、Gemini AI を活用して日本語の単語を瞬時に深層解析し、高品質な Anki カードを作成するためのオートメーションツールです。

### 主な機能 (Key Features)
* **深層構造解析**: 単語の読み、語源、意味、活用タイプ、そして自然な例文を AI が自動生成します。
* **音声自動合成**: 日本語のネイティブに近い発音データを自動的に取得し、カードに埋め込みます。
* **Anki 同步**: 生成されたデータは Python スクリプト経由でデスクトップ版 Anki に一括インポート可能です。

### デモ操作手順 (Step-by-Step Demo)

1.  **単語入力**: 入力欄に「学生」または「食べる」と入力して Enter を押します。
2.  **AI 解析のシミュレーション**: AI が構造化データを生成する様子を確認します。
3.  **カードプレビュー**: 生成されたカードのスタイル（表面・裏面）を右側のパネルで確認できます。
4.  **インタラクション**: スピーカーアイコンをクリックして、発音（シミュレーション）を体験してください。

> **注意**: このデモ版では API キーを使用せず、あらかじめ用意されたデータを使用して動作をシミュレートしています。
